# Graphs DSA Study

This repository tracks my progress, notes, and solutions for Graph problems. 

## Structure

- `implementations/`: Core boilerplate for defining graphs using different approaches (Adjacency List, Adjacency Matrix, Node-based variants) in Python, Java, and Go.
- `patterns/`: Categorized algorithmic patterns for Graphs (BFS, DFS, Topological Sort, Union-Find, Shortest Path, MST, etc.).
- `problems/`: Solutions organized by platform (Algoexpert, CTCI, EPI, Others) and difficulty.
- `notes/`: Scratchpad for quick concepts, Big O notes, etc.

## Logging Progress

To automatically commit a file as your DSA tracker expects, use the provided `add_solution.sh` script (make sure it is executable):

```bash
./add_solution.sh problems/algoexpert/medium/cycle_in_graph.py "Added Cycle in Graph problem"
```
